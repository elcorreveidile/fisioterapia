## 🔧 Instrucciones de Uso

### Para Midjourney
1. Usa los prompts tal como están escritos
2. Añade `--no 2` para consistencia
3. Añade `--style raw` para mayor control
4. Para variaciones, añade `--seed 12345` y cambia el número

### Para DALL-E 3
1. Simplifica el prompt eliminando especificaciones técnicas exactas de píxeles
2. Mantién el estilo y la descripción visual
3. DALL-E 3 puede necesitar prompts más descriptivos

### Para Generales (Stable Diffusion, etc.)
1. Divide prompts complejos en partes más pequeñas
2. Genera a mayor resolución y redimensiona
3. Añade especificaciones de color en formato HEX cuando sea crítico

## 💡 Consejos de Generación

1. **Consistencia:** Usa siempre los mismos colores específicos
2. **Estilo:** Mantén el estilo editorial limpio, sin excesos
3. **Profesional:** Todo debe transmitir confianza y profesionalidad
4. **Seguridad:** Las imágenes médicas deben ser seguras y respetuosas
5. **Prueba:** Genera varias opciones y selecciona la mejor

## 🎨 Referencias Visuales

**Estilo Inspiración:**
- Revistas médicas editoriales (The Lancet, NEJM)
- Marcas de salud modernas y minimalistas
- Editorial design de los años 50-60
- Tipografía clásica combinada con layout moderno

**Evitar:**
- Estilo corporativo genérico
- Imágenes de stock excesivamente "médicas"
- Colores brillantes o neón
- Tipografía sans-serif en todos los textos
- Iconografía compleja o redundante
